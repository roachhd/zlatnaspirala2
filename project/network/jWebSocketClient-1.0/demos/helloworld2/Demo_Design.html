Texts

Demo Description:

This little „Hello World“ demonstrates how to easily logon to the jWebSocket Server, broadcast a message and logoff again.
Please use the browser‘s show frame source function to see the entire source code of this page.

Short descriptions of the tabs inside the functional box:

  • Tokens:
    jWebSocket supports tokens, cross-platform compatible data objects, which can be exchanged between the WebSocket server and all kinds of clients, stationary and mobile, web based and native.
    Place your mouse on the buttons below to get more details.
  • Functions:
    jWebSocket supports data exchange as well as server side listeners and event handlers, bidirectional remote procedure calls and a secure access to SQL and NoSQL databases.
    Place your mouse on the buttons below to get more details.
  • Files:
    jWebSocket  supports file access and file sharing. You can easily save and load files to and from the server and load public or protected private directory content.
    Place your mouse on the buttons below to get more details.

Tooltips for buttons inside the functional box:

Under the "Tokens" tab (3 buttons):

  • Complex Object: Shows how to pass an arbitrary complex and hierarchical data object to the client.
  • Get Time: Shows how to simply pass the current server date and time token to the client.
  • Tokenizable: Shows how to serialize any Java object into a token and pass it to the client.

Under the "Functions" tab (3 buttons):

  • Listener: Shows how to pass an arbitrary complex and hierarchical data object to the client.
  • RPC: Shows how to simply pass the current server date and time token to the client.
  • Select: Shows how to serialize any Java object into a token and pass it to the client.

Under the "Files" tab (3 buttons):

  • Save file: Shows how to save a file in the public storage of the WebSocket server.
  • Load file: Shows how to load a file from the public storage of the WebSocket server.
  • Filelist: Shows how to list all available files in the public storage of the WebSocket server.
